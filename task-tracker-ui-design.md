## 1. 设计规范（配色与风格）

### 1.1 设计风格

- **主题：** 暗色主题（Dark Mode），参考 Linear / Notion / Jira 风格
- **风格关键词：** 简洁、专业、高效、数据密度适中、减少视觉噪音
- **圆角：** 6px（卡片/按钮）、4px（标签/徽章）、2px（输入框）
- **阴影：** 仅在弹出层/模态框使用 `box-shadow: 0 8px 32px rgba(0,0,0,0.4)`

### 1.2 色彩系统

#### 基础色板

```
背景色（Background）
├── --bg-base:       #0D0D0F   /* 最底层页面背景 */
├── --bg-surface:    #16161A   /* 卡片/面板背景 */
├── --bg-elevated:   #1E1E24   /* 悬浮/高亮层背景 */
├── --bg-hover:      #25252C   /* 悬停态背景 */

边框（Border）
├── --border:        #2A2A32   /* 默认边框 */
├── --border-light:  #3A3A44   /* 浅边框 */

文字（Text）
├── --text-primary:  #FFFFFF   /* 主标题/重要文字 */
├── --text-secondary:#A0A0B0   /* 正文/说明文字 */
├── --text-muted:    #5A5A6E   /* 次要/禁用文字 */
├── --text-inverse:  #0D0D0F   /* 深色背景上的文字 */

主题色（Primary / Accent）
├── --primary:       #5B7FFF   /* 主操作按钮、主链接（参考 SOP 标帆蓝）*/
├── --primary-hover: #7A9AFF   /* 主色悬停 */
├── --primary-muted: #2A3A7A   /* 主色背景（低饱和度）*/

语义色（Semantic）
├── --success:       #3DD68C   /* 已完成/成功 */
├── --success-muted: #1A3D2A   /* 成功背景 */
├── --warning:       #FFB020   /* 警告/进行中 */
├── --warning-muted: #3D2A0A   /* 警告背景 */
├── --danger:        #FF4D4F   /* 阻塞/错误/删除 */
├── --danger-muted:  #3D1A1A   /* 危险背景 */
├── --info:          #36CBCF   /* 信息/里程碑 */
├── --info-muted:    #1A3D3D   /* 信息背景 */
```

#### 状态色

```
任务状态色彩
├── 待处理（Todo）:    #5A5A6E（灰色） + --bg-elevated 背景
├── 进行中（In Progress）: #FFB020（橙色） + --warning-muted 背景
├── 已完成（Done）:     #3DD68C（绿色） + --success-muted 背景
├── 已阻塞（Blocked）:  #FF4D4F（红色） + --danger-muted 背景

优先级色
├── P0 / 紧急:  #FF4D4F（红色）
├── P1 / 高:    #FF7A45（橙色）
├── P2 / 中:    #FFB020（黄色）
├── P3 / 低:    #5A5A6E（灰色）
```

#### 甘特图专用色

```
任务条配色（按状态）
├── 进行中:  #FFB020（橙色）
├── 已完成:  #3DD68C（绿色）
├── 已阻塞:  #FF4D4F（红色）
├── 未开始:  #3A3A44（灰色）

里程碑:     #36CBCF（青色），菱形标记
依赖箭头:   #5B7FFF（蓝色），带箭头连接线
今日线:     #FF4D4F（红色），垂直虚线
```

### 1.3 字体规范

```
字体族：
├── 中文: "PingFang SC", "Microsoft YaHei", sans-serif
├── 英文/数字: "Inter", "SF Pro Display", sans-serif
├── 等宽（代码/时间戳）: "JetBrains Mono", "Fira Code", monospace

字号梯度：
├── 页面大标题:    20px / line-height: 1.3 / font-weight: 600
├── 卡片标题:      14px / line-height: 1.4 / font-weight: 600
├── 正文:          13px / line-height: 1.5 / font-weight: 400
├── 辅助说明:      12px / line-height: 1.4 / font-weight: 400
├── 标签/徽章:     11px / line-height: 1   / font-weight: 500
└── 数字/KPI:      24px / line-height: 1.2 / font-weight: 700
```

### 1.4 间距系统（8px 基准）

```
间距单位：4px 递增
├── 2px   （--space-xs）
├── 4px   （--space-sm）
├── 8px   （--space-md）
├── 12px  （--space-lg）
├── 16px  （--space-xl）
├── 24px  （--space-2xl）
├── 32px  （--space-3xl）
└── 48px  （--space-4xl）
```

### 1.5 动效规范

```
过渡时长：
├── 微交互（hover/focus）: 150ms ease
├── 展开/收起:           200ms ease-out
├── 模态框出现:          250ms ease-out（配合 scale 0.95→1）
├── 拖拽反馈:            100ms ease（实时）

动效原则：
├── 不做多余装饰动画
├── 拖拽时目标元素 scale(1.02) + 阴影加深
├── 任务条拖拽时显示半透明占位符
├── 列表项悬停背景色平滑过渡
```

### 2.1 应用 Shell

```
┌─────────────────────────────────────────────────────────────────┐
│  顶部导航栏（56px）                                              │
│  [Logo] [团队名称 ▾]     [🔔 3] [搜索] [头像 + 名字 ▾]           │
├────────┬────────────────────────────────────────────────────────┤
│        │                                                        │
│ 侧边栏  │                    主内容区                            │
│ (220px) │                                                        │
│        │                                                        │
│ [甘特图]│                                                        │
│ [成员]  │                                                        │
│ [看板]  │                                                        │
│ [日志]  │                                                        │
│        │                                                        │
│ ────── │                                                        │
```

**侧边栏交互：**
- 收起态（图标模式）：宽度 56px，只显示图标
- 展开态（默认）：宽度 220px，图标+文字
- 切换按钮：侧边栏底部 `‹` / `›`

**顶部栏细节：**
- 搜索：`Cmd/Ctrl + K` 全局快捷键触发
- 通知铃铛：红点徽章显示未读数
- 头像下拉：切换主题/退出登录

### 3.2 核心元素

| 元素 | 描述 |
|------|------|
| **时间轴头** | 显示日期/周，支持日/周/月三档切换 |
| **任务树** | 左侧树形结构，支持折叠/展开子任务（缩进 16px/层级） |
| **任务条** | 横向彩色条，根据起止日期对齐时间轴 |
| **里程碑** | 菱形 ◆ 标记，颜色 `--info` |
| **依赖箭头** | 蓝色曲线箭头，连接前置任务终点 → 后置任务起点 |
| **今日线** | 红色垂直虚线，当前日期位置 |
| **进度条** | 底部全局进度指示器 |
| **项目分组** | 按项目折叠的任务组，顶层节点为项目名 |

### 3.3 甘特图交互细节

#### 任务条拖拽

```
操作              结果
─────────────────────────────────────────────────────────────
拖拽任务条左右边缘   调整开始/结束日期（光标变为 ↔）
整体拖拽任务条       移动任务的开始/结束日期（保持时长不变）
双击任务条           打开任务详情弹窗
右击任务条           上下文菜单（编辑/删除/添加依赖/设为里程碑）
悬停任务条           显示 tooltip（任务名/负责人/起止日期/完成度）

拖拽约束：
├── 不得早于前置任务结束时间（显示红色警告线）
├── 超出项目范围时自动吸附到边界
└── 拖拽时任务条半透明（opacity: 0.7），下方显示半透明占位条
```

#### 时间轴缩放

```
缩放级别     每格宽度    适用场景
──────────────────────────────────
日视图       40px/天     短期排期/当日查看
周视图       80px/周     中期计划/本周全局
月视图       24px/周     长期规划/季度视图

交互：
├── 鼠标滚轮 + Ctrl/Cmd → 缩放时间轴
├── 左侧滑块 → 拖拽调整缩放比例
├── 双指捏合（触控板）→ 缩放
└── 左右拖拽空白区 → 滚动时间轴
```

#### 依赖箭头绘制

```
依赖关系类型：
├── FF（Finish-to-Finish）：前置任务结束 → 后置任务结束
├── FS（Finish-to-Start）：前置任务结束 → 后置任务开始（最常用）
└── SS（Start-to-Start）：前置任务开始 → 后置任务开始

箭头样式：
├── 线型：贝塞尔曲线，带箭头
├── 颜色：#5B7FFF（主色）
├── 悬停：线宽 2px → 3px，箭头放大
└── 悬停前置/后置任务时，高亮相关联的所有箭头

新增依赖：
├── 右击任务A → "添加依赖" → 选择任务B
└── 自动判断 FS 关系
```

### 3.4 甘特图组件清单

| 组件 | 说明 |
|------|------|
| `GanttTaskBar` | 任务条，支持拖拽 resize/移动 |
| `GanttMilestone` | 里程碑菱形标记 |
| `GanttDependencyArrow` | 依赖关系箭头 |
| `GanttTodayLine` | 今日线（红色虚线） |
| `GanttTimeHeader` | 时间轴表头（日期网格） |
| `GanttTaskTree` | 左侧任务树（折叠/展开） |
| `GanttZoomSlider` | 缩放控制条 |
| `GanttContextMenu` | 右键上下文菜单 |
| `GanttTooltip` | 悬停提示气泡 |
| `GanttAddTaskModal` | 新建/编辑任务弹窗 |

## 8. 交互规范

### 8.1 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + K` | 全局搜索（任务/成员）|
| `Cmd/Ctrl + N` | 新建任务 |
| `G + G` | 跳转甘特图 |
| `G + M` | 跳转成员视图 |
| `G + B` | 跳转看板 |
| `G + L` | 跳转日志 |
| `Esc` | 关闭弹窗/抽屉 |
| `←` `→` | 日志日期切换 |

### 8.2 通知与反馈

```
操作反馈：
├── 任务状态变更 → Toast: "已移动到 [进行中]"
├── 保存成功 → Toast: "保存成功"
├── 操作失败 → Toast: "操作失败，请重试"（红色）
├── 拖拽释放 → 任务条平滑动画到新位置（200ms）
└── 删除确认 → 模态框二次确认

网络状态：
├── 请求中 → 按钮显示 spinner
├── 加载中 → 骨架屏（Skeleton）
└── 加载失败 → 空状态 + 重试按钮
```

### 8.3 空状态设计

| 场景 | 空状态插画+文案 |
|------|----------------|
| 甘特图无任务 | 📊 + "还没有任务，创建一个开始规划吧" + [新建任务] |
| 看板无任务 | 📋 + "此列还没有任务" + [添加任务] |
| 成员视图无成员 | 👥 + "团队还没有成员" + [邀请成员] |
| 日志无记录 | 📝 + "今日还没有人填写工作日志" + [立即填写] |

### 8.4 响应式断点

```
断点          宽度         布局变化
────────────────────────────────────────────────
桌面          ≥ 1280px     完整布局（侧边栏+主内容）
平板          768-1279px   侧边栏收起为图标模式
手机          < 768px       底部 Tab Bar，侧边栏隐藏，全屏页面
```
